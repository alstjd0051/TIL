# 명령어 모음집

> 1. 탐색하기
>
> - ls
>   - 현재 경로에 존재하는 파일/폴더 확인하기(list)
> - pwd
>   - 현재위치 경로 출력하기 (print working directory)
> - cd
>   - 디렉토리 이동하기 (change directory)
>
> 2. 파일/폴더 관리하기
>
> - touch
>   - 파일생성하기
> - mkdir
>   - 폴더생성하기 (make directory)
> - cat
>   - 파일 확인하기(concatenate)
> - rm
>   - 파일 삭제하기(remove)
> - rmdir
>   - 폴더 삭제하기(remove directory)
> - rm -r
>   - 파일을 갖고 있는 폴더 삭제하기 (remove -remove)
> - cp
>   - 복사하기(copy)
> - mv
>   - 파일/폴더 이동시키기, 이름 변경하기
>
> 3. 기타 유용한 명령어
>
> - clear
>   - 터미널 정리하기
> - history
>   - 이전에 사용한 명령어 확인하기
> - man
>   - 명령어 메뉴얼 확인하기
